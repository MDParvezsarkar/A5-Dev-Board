Question-1: What are the different ways to select an element in the DOM?

There are 6 different ways to select an element in the DOM.
1.querySelector() 2.querySelectorAll() 3.getElementById() 4.getElementsByClassName() 5.getElementsByTagName() 6.getElementsByName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is innerHTML sets the HTML content inside an element but innerText sets only the visible text inside an element, and textContent sets all text inside an element.

Question-3:What is event delegation in the DOM?

Event Delegation is a technique where instead of attaching event listeners to individual child elements, we attach a single event listener to a parent element. When a child triggers the event, it bubbles up to the parent, which then determines which child element was clicked or interacted with.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM where an event triggered on a child element first executes on that element and then propagates (bubbles up) through its parent elements, all the way to the root.
